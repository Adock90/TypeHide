# TypeHide
A program for writing sensitive data in plaintext to a file whilst hiding when typing

No dependancies needed as libaries come with python except on linux you will need to run:
<code>sudo apt install python3-tk #for debian/ubuntu based systems</code>
or for fedora/red hat based systems
<code>sudo dnf install python3-tkinter</code>
or your distro package manager

then run with:
<code>python3 HideType.py</code>
